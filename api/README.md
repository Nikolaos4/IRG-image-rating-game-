## Как запустить

1. Установите зависимости:
    ```
    npm i
    ```
2. Введите переменные окружения в `.env` файле. Вы можете использовать `.env.example` как шаблон.
3. Выполните миграцию базы данных:
    ```
    npx prisma migrate dev
    ```
4. Сгенерируйте Prisma Client:
    ```
    npx prisma generate
    ```
5. Заполните базу начальными данными
    ```
    npm run seed
    ```
6. Запустите сервер:
    ```
    npm run dev
    ```
